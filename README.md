LAVA_RGB-TO-PPU_LITE
This project converts the existing Lava RGB board into a fully functional and self-contained FPGA PPU chip that doesn't require the original RP2S02 PPU chip. It requires removing the comparators from the board and soldering several jumpers to enable the local PPU bus.
